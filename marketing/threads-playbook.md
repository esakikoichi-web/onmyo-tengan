# Threads集客プレイブック（青龍孔一）

実際の鑑定を"実話"として淡々と流し、**「人生の決断の前にいる、深く視てほしい人」だけ**を
公式LINE（@genri__gensoku）に呼び込むための型。

前提：
- 変換（LINEでの説明→予約）は既に完成している。**問題はトップ＝入口の人数と"質"だけ。**
- 狙うのは少数の"正しい人"。バズって数万人集める必要はない。5.5万を払う人が月に数人増えれば十分。
- あの界隈がAIと作り話で溢れてるからこそ、**本物の実話は逆に刺さる。**

---

## 絶対ルール（これを守る限り、気持ち悪くならない）

1. **実話だけ。作り話をしない。**
2. **必ず匿名＋本人の許可を取る。**（「匿名で、誰かの役に立てさせてもらっていい？」でほぼOKをもらえる）
3. 年代・性別・職種は**ぼかす**。特定される情報は出さない。（例：「40代・独立を考える男性」）
4. 「必ず当たる」等の**誇張をしない**。
5. **生年月日の大量収集で釣らない**（＝タダ狙いを避け、質の高い人だけ残す）。

---

## 実話ポストの型（6ブロック）

| ブロック | 役割 | 書き方 |
| --- | --- | --- |
| ①フック（1行目） | スクロールを止める | 具体的な一場面。「先週、◯代の△△が来た。□□するか、ずっと迷っていた。」 |
| ②葛藤 | 読者に"自分だ"と思わせる | その人が本当に怖がっていたこと |
| ③鑑定で起きたこと | 価値の提示 | 当てた話ではなく「全部聞いた・現在地が見えた」 |
| ④変化 | 結果 | 帰る時どう変わったか（本人の言葉があれば引用） |
| ⑤学び | 読者が自分に当てはめる | 1〜2行で一般化 |
| ⑥（時々）静かな一言 | 導線 | 「深く視てほしい人は、プロフィールから。」※3〜4投稿に1回だけ |

> **Threadsは1行目（フック）で決まる。** ここに一番力を入れる。

---

## CTA（売らない。説得力で流す）

- 基本は**売り込まない**。投稿の中身で、プロフィール→LINEに自然に流れる。
- 静かな一言は**3〜4投稿に1回だけ**。「深く視てほしい人は、プロフィールから。」
- **リプ欄で会話する**（Threadsはリプで伸びる）。個別の悩みに短く反応→深い話は「よければLINEで」。
- 「生年月日ください」で釣らない。

---

## ネタ切れしない仕組み（あなたの武器＝毎回の鑑定）

鑑定が終わるたびに、**メモ3行**を残す：
1. 相談者の状況（匿名・何に迷って来たか）
2. 視えた／伝えたこと
3. 帰る時、どう変わったか

→ 今使っているスプレッドシートに「ネタ帳」列を足すだけ。週末に3行メモ→投稿に整える。
**毎回の鑑定が、そのまま次の投稿になる。** これが競合（作り話・AI量産）には出せない差。

---

## 頻度と運用

- 実話 or 観察を**週3〜4本**。**継続が最優先**（バズより積み重ね）。
- 各投稿はリプにちゃんと返す＝アルゴが伸ばす＆信頼が貯まる。
- 同じ内容を**IGストーリーにも流用**（既にやっている質問箱の基盤を活かす）。
- たまに「観察・哲学」系（下記）を混ぜて人柄を見せる。

## すぐ投稿できる実例

### A. 実話（まーくん／独立）※本人の許可を取ってから
```
先週、ひとりの男性が来た。

植物屋を始めたい。でも「本当に食っていけるのか」と、
ずっと踏み出せずにいた。

盤を視て、いちばん話したのは占いの吉凶じゃない。
彼の性質、家族のこと、何を怖がっているのか。
90分、全部聞いた。

帰る頃、彼は言った。
「せっかくのチャンスを、無駄にしません」

当てたわけじゃない。
見えなくなっていた"自分の現在地"が、見えただけ。

人が迷うのは、能力が足りないからじゃない。
今どこにいるかが、見えなくなっているだけのことが多い。
```

### B. 観察／positioning（作り話ではない＝あなたの考え。そのまま使える）
```
当たる占いを、僕は信用していない。

未来を言い当てて驚かせることと、
その人が前に進めることは、まったく別の話だから。

僕がやるのは前者じゃない。
90分かけて、あなたが"もう迷わない"状態を作る。
それだけ。
```

### C. 観察／運の話（青龍孔一の芯：「運は自分で作るもの」）
```
「運が良くなりますように」

そう願っている限り、運は良くならない。

運は、天から降ってくるものじゃない。
自分で作るものだ。

いつ動くか。何を選ぶか。誰と組むか。
その一つひとつが、運を形づくっていく。

僕がやるのは、運を"当てる"ことじゃない。
あなたが今どこにいて、どこへ動けるのかを、一緒に視る。
運を作るのは、いつだって、あなた自身だ。
```

### 実話の"型"だけ（あなたの実際の鑑定で埋める）
```
先週、〔◯代・△△（職種はぼかす）〕が来た。
〔□□するか〕、〔ずっと/何年も〕迷っていた。

本当に怖がっていたのは〔本当の葛藤〕。

盤を土台に、〔仕事・家族・お金〕まで全部聞いた。
当てる話は、ほとんどしていない。

帰る頃、〔本人の言葉 or 変化〕。

〔1〜2行の学び：迷いは能力の問題じゃない、等〕
```

## 立ち上げ：小さいアカウントからの"切り込み方"

フォロワーが少ないうちは、投稿を"打ち上げる"だけじゃ誰も見ない。**切り込む＝まずリプで殴り込む。**

1. **リプが最速の入口（毎日10〜15）**
   - 占い界隈の大きい投稿（tougarashi等、リプ千件クラス）に、短くて本物のリプを置く
   - 自己宣伝はしない。あなたの"視点"だけ置く（例：「当てるより、迷いを消す方が大事だと思う」「運は自分で作るもの」）
   - AIコピペのリプが並ぶ中で、本物の一言は逆に目立つ → プロフ訪問 → フォロー → LINE
2. **自分の投稿は1日1本**（実話 or 観察。1行目のフックに全力）
3. **固定投稿を1つ**：あなたが誰で、何をする人か（正直な自己紹介。他人の真似でなく、あなたの言葉で）。プロフ訪問者がLINEへ進む入口
4. **プロフィール（bio）**：一言＋LINE導線
5. **測り方**：最初はいいね数を見ない。**続けた日数・プロフ訪問・LINE追加**で測る。2週間走る。

> 一歩目：大きい占い投稿に、本物のリプを3つ置く。投稿ゼロでもいい。まずリプで"存在"を出す。

## やらないこと（再掲・大事）

- 作り話・盛った話 → 一度バレると全部が嘘になる
- 生年月日を大量に集める撒き餌 → タダ狙いばかり来る
- 「必ず当たる」系の煽り → あなたの"格"を下げる

**正直で・実話で・少数の正しい人に。** これがあなたの勝ち筋。
