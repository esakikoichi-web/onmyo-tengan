# 陰陽天眼 サブスク商品設計

> 「歩合営業の専属AI参謀。毎朝、あなたの客リストが『今日攻める順』に並び替わり、一人ひとりの攻略トークと商談吉日が届く。月額9,800円・完全自動。」

**スタイル：完全放置（人的サポートなし）。** 工数ゼロで無制限にスケールする代わり、
「毎日の自動価値」だけで¥9,800を証明しきる設計に全リソースを寄せる。

## 1. ポジショニング

- **占いではなく「営業の日次インテリジェンス」として売る。**
- 理由：占いは「一度観たら終わり」でサブスクと相性が悪い。本アプリの強みは
  今日の攻め順 / 商談吉日 / 客ごとの攻略トークという**毎日使う道具**であること。
- ターゲット：**高単価歩合の営業**（保険・不動産・車・太陽光・リフォーム等）。
  1件の歩合が数十万円の業種に絞る。月1万は「1件の0.1件分」でROIが桁で合う。
  小口・日雑貨営業には高すぎるので狙わない。

## 3. 無料 → 有料の壁

| 機能 | 無料（味見） | ¥9,800/月 |
|---|---|---|
| 自分の生年月日を観る | ◯ 1件だけ | ◯ 無制限 |
| 客の生年月日で攻略トーク | ✕ | ◯ 無制限 |
| 今日の攻め順・今週の吉日 | ✕ | ◯ 毎朝LINE配信 |
| 客リスト・履歴の保存 | ✕ | ◯ 端末越しに保存 |
| 勝負商談シート（自動の深掘り） | ✕ | ◯ 無制限 |

- 無料の形：**7日間トライアル（カード登録あり）＋味見1件**。
- フリーミアムにしない（中身がコピーされやすく価値が薄まるため）。

## 4. 完全放置モデルの継続エンジン（人的サポートなし）

人の手が無い分、**「毎日手放せない道具」に振り切る**ことで¥9,800を正当化する。
柱は3つ。すべて自動＝工数ゼロ。

### 柱1：毎朝、客リストが「今日攻める順」に自動で並び替わる
- 日付で吉凶が変わるため中身が毎日入れ替わる → 「昨日観たからもういい」が起きない。
- 営業が朝これを見て動く＝**業務プロセスに埋め込む**。習慣化＝解約されない。
- 実装：登録済み客リスト × 当日の暦で `rToday` / `bestSlot` 相当を毎朝再計算し、LINEへ自動配信。

### 柱2：新規客を入れるたびに攻略トークが出る
- 営業は毎週新しい客に会う。会うたびにツールが要る＝**構造的に解約できない**。
- 放置モデル最大の継続エンジン。新規客フローが止まらない限り価値が続く。

### 柱3：勝負商談シート（自動の深掘り）
- 大口商談の客を入れると、通常より長く濃い攻略書を**自動生成**。
- 人の鑑定の"重厚感"を自動で代替する。無制限に使える。

### スケール（工数≒ゼロ）

| 契約者数 | 月商 | 追加工数 |
|---|---|---|
| 100人 | 98万円 | ほぼゼロ |
| 500人 | 490万円 | ほぼゼロ |
| 1,000人 | 980万円 | ほぼゼロ |

**天井が無いのが放置モデルの本質的な強み。** ソロで年商1億も構造上は可能。

### 解約対策（放置モデルの最重要課題）
1. **オンボーディングで7日以内に「1件の成功体験」を作る導線**（トライアル中に1商談でツールを使わせる）。
2. **年額プランで囲い込む**（月次解約を物理的に減らす）。
3. **客データの蓄積で乗り換えコストを上げる**（50人登録済み＝離れると痛い）。
4. 客登録数が増えるほど毎朝配信の価値が上がる設計。
